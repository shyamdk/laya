# Vision Document (v1.1)

## Vision
Build an AI-assisted adaptive learning platform that delivers personalized education while preserving sound pedagogy.

## Long-term Goals
- Support multiple grades and subjects.
- Use AI to explain, motivate and personalize.
- Measure mastery rather than memorization.
- Provide parents and teachers with meaningful insights.

## Core Pillars
1. Learning Science
2. Adaptive Engine
3. AI Tutor
4. Gamification
5. Analytics
6. Accessibility
7. Privacy & Security

See ADR for architectural rationale.
