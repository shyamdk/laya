# MVP Specification (v1.1)

Goal: Validate the complete platform using Multiplication Tables.

Features:
- Authentication
- Student profile
- Learn mode
- Adaptive timed tests
- Progress dashboard
- Mastery tracking
- AI explanations
- AI coach
- Parent session summary
- Flutter (Android/iOS/Web)
- Supabase/PostgreSQL backend

Success Criteria:
- End-to-end user journey works.
- Adaptive algorithm updates mastery.
- AI explanations improve review experience.
- Architecture reusable for future subjects.

Out of Scope:
- Teacher portal
- Parent portal
- Voice tutor
- Multi-grade curriculum
